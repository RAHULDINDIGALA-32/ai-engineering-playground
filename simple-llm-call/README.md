# Simple LLM Call

A small Python example that sends a prompt to the Groq API and prints the model response.

## Setup
1. Install dependencies:
   `pip install -e .`
2. Create a `.env` file and add your Groq API key:
   `GROQ_API_KEY=your_api_key_here`
3. Run the script:
   `python hello_llm.py`

## Notes
- This project uses the `groq` and `python-dotenv` packages.
- It demonstrates a basic LLM request with a simple prompt.
